## 1. Application Overview

The Warranty Management Mobile App is designed to streamline the complete warranty lifecycle for Maruthi Motor Pump products — from manufacturer stock entry to end-consumer warranty registration and tracking.

### Key Objectives:
- Digitize warranty registration with photo proof
- Track product flow: Manufacturer → Distributor → Dealer
- Auto-calculate warranty expiry and status
- Provide role-based access for different stakeholders
- Real-time dashboard with business insights

---

## 2. User Roles & Access Control

| Role | Access Level |
|------|-------------|
| **Admin** | Full access — manage users, products, all stock levels, all warranties, dashboard |
| **Manufacturer** | Add products, manage manufacturer stock, view dashboard |
| **Distributor** | Receive stock from manufacturer, distribute to dealers |
| **Dealer** | Register warranties, manage dealer stock, add consumers |

#### Level 2: Distributor Stock
| Field | Description |
|-------|-------------|
| From Manufacturer Stock | Select source batch |
| Product | Product received |
| Quantity | Units received |
| Received Date | Date of receipt |

#### Level 3: Dealer Stock
| Field | Description |
|-------|-------------|
| From Distributor Stock | Select source |
| Product | Product received |
| Model | Model number |
| Invoice Number | Purchase invoice |
| Date | Receipt date |

**Stock Flow:**
```
Manufacturer → Distributor → Dealer → Consumer (via Warranty Registration)
```
